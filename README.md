# HR-Manager

## Vorwort
Sehr geehrter Leser,  
Vielen Dank für Ihr Interesse an der von mir entwickelten Software.  

Die Idee für ein HR-Manager-Tool entstand während unserer Projektarbeit im Rahmen des Lehrplans der LuG ACADEMY. Dieses Tool wurde entwickelt, um die Verwaltung von Mitarbeitern und firmeninternen Prozessen zu erleichtern.  

### Funktionen des Tools:
- Verwaltung neuer und bestehender Mitarbeiter.
- Anzeige und Versendung von kritischen firmeninternen Nachrichten.
- Rollenbasierte Zugriffskontrolle, die von Administratoren festgelegt wird.
- Grafische Konsolenanmeldung mit menübasierter Darstellung, abhängig von der Benutzerrolle.
- Speicherung und Abfrage von Daten über JSON-Dateien als Schnittstelle.
